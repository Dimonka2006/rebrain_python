
Функция 	Описание
sequ[ind]	Возвращает элемент последовательности sequ, стоящий на позиции ind
sequ[beg, end, step]	Возвращает срез последовательности, начиная с элемента номер beg и 
заканчивая end-1 с шагом step
sequ1 + sequ2	Возвращает последовательность, представляющую собой конкатенацию sequ1 и sequ2
sequ * n	Возвращает последовательность, состоящую из повторяющейся n раз последовательности sequ
len(sequ)	Возвращает длину последовательности sequ
sequ.index(el)	Возвращает индекс первого элемента cо значением el в последовательности sequ
sequ.count(el)	Возвращает количество элементов со значением el в последовательности sequ
